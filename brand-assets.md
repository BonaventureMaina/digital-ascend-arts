## Logo System

### Primary Logo (Horizontal)
- **Usage:** Default for all applications
- **Minimum Width:** 180px (digital) / 1.5" (print)
- **Clear Space:** Icon height on all sides
- **Components:** Gradient icon + "Digital Ascend" + "ARTS PRODUCTION"

### Logo Variations
1. **Standard (Light backgrounds)** - Dark gradient icon + dark text
2. **Inverted (Dark backgrounds)** - Light gradient icon + light text
3. **Icon Only** - For social avatars, favicons, app icons
4. **Wordmark Only** - Restricted use when icon cannot be included

### Icon Specifications
- **Geometry:** 4 ascending bars with gradient
- **Gradient Direction:** Bottom to top (dark to light)
- **Aspect Ratio:** Square (1:1)
- **Export Sizes:** 16px, 32px, 64px, 128px, 256px, 512px

## Color System - Contemporary Edge

### Primary Colors
```
Deep Black     #0f1419  - Primary backgrounds, dark mode base
Charcoal       #1a1a1a  - Headlines, logo, high contrast text
Midnight Blue  #1e3a5f  - Primary accent, CTAs, links
Steel Blue     #4a7ba7  - Interactive elements, hover states
Ice            #e8eef3  - Light backgrounds, subtle surfaces
White          #ffffff  - Text on dark, card backgrounds
```

### Color Roles
- **Backgrounds (Light Mode):** Ice (#e8eef3), White (#ffffff)
- **Backgrounds (Dark Mode):** Deep Black (#0f1419), Charcoal (#1a1a1a)
- **Text on Light:** Charcoal (#1a1a1a), Midnight Blue (#1e3a5f)
- **Text on Dark:** White (#ffffff), Ice (#e8eef3)
- **Primary Actions:** Midnight Blue (#1e3a5f)
- **Hover/Active:** Steel Blue (#4a7ba7)

### Accessibility
- All text combinations meet WCAG AA standards minimum
- Primary text: Charcoal on White = 14.8:1
- Link text: Midnight Blue on White = 8.2:1

## Typography System

### Primary Typeface: Space Grotesk
**Source:** Google Fonts  
**Weights:** 400 (Regular), 500 (Medium), 600 (Semibold), 700 (Bold)

**Usage:**
- Headlines: 600 (Semibold) or 700 (Bold)
- Subheadings: 500 (Medium) or 600 (Semibold)
- Body Text: 400 (Regular)
- Emphasis: 500 (Medium) or 600 (Semibold)
- UI Elements: 500 (Medium)

### Type Scale
```
Hero Display:     48-72px / 3-4.5rem
H1 (Page Title):  36-48px / 2.25-3rem
H2 (Section):     28-32px / 1.75-2rem
H3 (Subsection):  22-24px / 1.375-1.5rem
H4 (Component):   18-20px / 1.125-1.25rem
Body Large:       18px / 1.125rem
Body Standard:    16px / 1rem
Body Small:       14px / 0.875rem
Caption:          12px / 0.75rem
```

### Line Heights
- Headlines: 1.1-1.2
- Body Text: 1.6-1.7
- UI Elements: 1.4-1.5

### Letter Spacing
- Large Headlines (48px+): -0.02em
- Standard Headlines: 0em
- Body Text: 0.01em
- All Caps (UI/Tags): 0.05-0.1em

## Logo Usage Guidelines

### DO
✓ Maintain minimum clear space  
✓ Use approved color combinations  
✓ Scale proportionally  
✓ Ensure sufficient contrast  
✓ Use vector formats (SVG)  
✓ Use icon-only for social avatars  

### DON'T
✗ Stretch or distort  
✗ Change gradient colors/direction  
✗ Add effects (shadows, glows)  
✗ Place on busy backgrounds  
✗ Use below minimum size  
✗ Rotate or skew
